# Week 13 - Activity 5: LLM Diet & Exercise Planner

## How it works
The app takes a user's BMI, age, and gender and sends them to an LLM (Groq/OpenAI).
The LLM generates a one-month personalised diet and exercise plan based on the inputs.

## Scenario 1: Young Underweight Person
- BMI: 16.5, Age: 20, Gender: Female
- The LLM detected underweight status and generated a high-calorie diet with strength training.

## Scenario 2: Older Overweight Person
- BMI: 31.0, Age: 55, Gender: Male
- The LLM detected overweight status and generated a calorie-deficit diet with low-impact cardio.
